# OMNI Voice — Distribution

Canal público de distribuição do app **OMNI Voice** (transcrição de voz nativa para macOS).

- **Última versão:** confira a aba [Releases](https://github.com/InovacaoRD/omni_voice_dist/releases/latest).
- **Auto-update:** Sparkle, configurado para checar este repositório.
- **Código-fonte:** privado, em `InovacaoRD/omni_voice`.

## Instalação

1. Baixe `OMNI Voice.dmg` da release mais recente.
2. Arraste para `Applications/`.
3. Abra. Cole sua chave de ativação no primeiro launch.
4. Aceite as permissões pedidas (Microfone, Acessibilidade, opcionalmente Gravação de Tela).
5. Aperte `⌘R`, fale, solte. Texto cola onde estiver focado.
